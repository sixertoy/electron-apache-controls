# Electron Apache Controls

## Install

```bash
git clone https://github.com/sixertoy/electron-apache-controls.git
yarn install
```
## Run in dev mode

```bash
yarn dev
```

## Build your own

```bash
git clone https://github.com/sixertoy/electron-apache-controls.git
yarn install
yarn build
yarn compile
```
